Flashcard Study Application
 Project Link

Developed and designed a flashcard study application using Java, PostgreSQL, and Vue.js involved a comprehensive approach that seamlessly integrated the strengths of each technology stack.

The backend, powered by Java and Spring Boot, provides a robust foundation for user authentication, flashcard management, and database interaction.

Leveraging PostgreSQL ensures a reliable and scalable data storage solution, allowing users to create, edit, and organize their flashcards effectively.

Meanwhile, the Vue.js frontend delivers a dynamic and responsive user interface, enabling an interactive learning experience. With Vue Router for navigation and Vuex for state management, the frontend seamlessly integrates with the Java backend through RESTful APIs, ensuring smooth communication and efficient data flow.
